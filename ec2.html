ec2.py¶

Part C cf schema

ec2.main(region: str, table_name: str, batch_size: int, bucket_in: str, bucket_out: str, annotator_names: list) → None¶

Get all image without annotator from dynamoDB. Generate json via with annotation by batch and upload to s3. update dynamoDB with annotator name.

Parameters

  • region – dynamoDB table region

  • table_name – dynamoDB table name

  • batch_size – batch size for via json

  • bucket_in – bucket with standardised image

  • bucket_out – bucket where via json are upload

  • annotator_names – list the name of all the annotator
